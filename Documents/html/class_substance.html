Detailed Description

The substance class is a type of interface (base class) that provides the basic necessary functionality for using OpenGL shaders. The class is designed to be expandable so that you can override certain methods and functionality to better fit the requirments of any custom written shaders. An instance of the 'Substance' class can be attached to a 'Mesh-Renderer' where the shader can be 'loaded', 'binded' and excuted during runtime.
